1) Para compilar el programa se debe ingresar a la terminal dentro de la carpeta del proyecto.
2) Dentro se escribe "make" para compilarlo.
3) Existen varias opciones al momento de iniciar el programa, se puede correr el programa con
lo siguientes formatos: 

   <prog> <file> <file-list>

   <prog> <file>

donde "file" es el archivo de frecuencias y "files-list" la lista de los documentos que va a 
procesar el programar.

4)El programa va indicar cuales archivos se leen.
5)Luego desplegara la siguiente tabla:

============================== Frequencies Table==============================
	Hex Code		 Symbol			 Frequency
==============================================================================
 
Indicando en cada columna respectivamente, numero hexadecimal, el simbolo(si es despegable) y la frecuencia.
